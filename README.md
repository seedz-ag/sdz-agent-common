# Sumário

Este pacote contém as funçoes comuns para o Cliente de Integração Seedz.

# Detalhes

Logger - Logs para aplicação
Validator - Validacão das configuracoes do client.
Hydrator - Responsável pelos DTOs.

## Pacotes dependentes:

- sdz-agent-types
- chalk
- moment
